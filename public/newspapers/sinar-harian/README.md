# Sinar Harian Newspaper Storage

This directory stores Sinar Harian newspaper front page images.

## Sample Image

A sample placeholder image (`sample-placeholder.jpg`) is provided for testing the upload and search functionality.

## Production Usage

Upload your Sinar Harian newspaper front pages here using the upload interface at `/newspaper-search`.

The application will:
- Extract text content using OpenAI Vision API
- Create searchable embeddings
- Store metadata in the database
- Make images accessible for search results
